# Win The Million (Кто хочет стать миллионером)

Простая консольная игра-викторина, написанная на Python.

## Как запустить

1. Убедитесь, что у вас установлен Python 3.
2. Скачайте файлы проекта.
3. Запустите игру командой:
   ```bash
   python milion.py
   ```

## Текущие возможности

- Демонстрация вопроса о столице Германии.
- Вывод вариантов ответов.
